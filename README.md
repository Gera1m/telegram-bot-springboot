# 🤖 Bot de Telegram con Spring Boot

Este proyecto implementa un bot de Telegram utilizando Spring Boot y la librería TelegramBots.

## 📌 Descripción

El bot responde a mensajes enviados por el usuario mediante el modelo Long Polling.
Fue desarrollado como práctica para la materia de Desarrollo de Aplicaciones.

## 🚀 Tecnologías utilizadas

- Java 11
- Spring Boot
- Maven
- TelegramBots API

## ⚙️ Funcionalidades

- Responde al comando `/start`
- Responde a mensajes enviados por el usuario
- Se ejecuta como aplicación Spring Boot

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio:
- git clone https://github.com/Gera1m/telegram-bot-springboot.git

2. Configurar el token del bot en la clase correspondiente.

3. Ejecutar como **Spring Boot App** desde el IDE.

## 🔒 Seguridad

El token del bot fue eliminado del repositorio por razones de seguridad.
Debe configurarse manualmente antes de ejecutar el proyecto.
